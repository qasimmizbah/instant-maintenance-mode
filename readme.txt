Quickly enable or disable maintenance mode for your WordPress site with custom logo.

== Description ==

Instant Maintenance Mode allows you to temporarily take your WordPress site offline for maintenance while showing a professional maintenance page to visitors.

**Features:**
- One-click maintenance mode toggle
- Upload custom logo for maintenance page
- Customizable maintenance message
- Admin users can still access the site
- Mobile-responsive design
- Easy to use with clean interface

Perfect for when you need to perform site updates, maintenance, or other changes without visitors seeing a broken site.

== Installation ==

1. Upload the `instant-maintenance-mode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Instant Maintenance' in the admin sidebar to configure

== Frequently Asked Questions ==

= How do I enable maintenance mode? =
Go to Instant Maintenance in your WordPress admin sidebar, check the "Enable Maintenance Mode" option, and save changes.

= Can administrators still access the site? =
Yes, any user with 'manage_options' capability (typically administrators) can still access the site normally.

= Can I customize the maintenance page? =
Yes! You can upload a logo, set custom text.
